# Description

1. Creating a method that calculates the area of a rhombus.
2. Creating a method that takes three numbers, outputs and returns the smallest.
3. Creating a method that outputs an array.
4. Creating a method that returns the largest number from the array.

Instructions for creation
Clone the repository to your local machine using

# Instructions for starting
To start the project: when we have already cloned the project, open the terminal and write the following commands
```
cd src
```
And we write the commands to launch the project
```
javac Main.java

java Main
```
